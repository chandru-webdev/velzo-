# Velzo

Live Demo: https://velzo-three.vercel.app/html/index.html

## 📌 Overview

Velzo is a **fashion e-commerce landing website** built with HTML/CSS/JS showcasing collections, styles, and seasonal fashion products.  
The goal of the project is to create a responsive UI for a modern streetwear store.

> ⚠️ **Work in progress:** This project is not fully complete yet.  
I’m actively developing features, improving responsiveness, and adding more sections.

##  Features

-  Home section with top products  
-  Collections section  
-  Wedding store showcase  
-  Winter wear collection  
-  New arrivals  
-  Responsive layout (in progress)

## Built With

This project uses:

- HTML  
- CSS  
- JavaScript (if applicable)
